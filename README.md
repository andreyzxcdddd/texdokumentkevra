SQLite3 на Windows: установка и запуск
Учебный проект, который шаг за шагом показывает, как установить и запустить консольную программу `sqlite3` на Windows, а также выполнить простейшие команды для работы с базой данных.
 Описание проекта
Этот проект создан для студентов, которые только начинают знакомство с базами данных и SQLite.
В рамках проекта вы научитесь:
- скачивать SQLite3 для Windows;
- распаковывать архив и запускать `sqlite3.exe` через командную строку;
- создавать простую базу данных и таблицу;
- выполнять простые SQL-запросы (создание, вставка и просмотр данных).
Проект предназначен для:
- студентов колледжа и вузов;
- начинающих разработчиков, которые хотят быстро попробовать SQLite на Windows без сложных настроек.

Установка и запуск SQLite3 на Windows
Ниже приведена простая пошаговая инструкция.
1. Скачать SQLite для Windows
1. Открыть браузер (Chrome, Edge и т.п.).
2. Перейти на официальный сайт SQLite (страницу загрузки для Windows).
3. Скачать архив вида:  
   `sqlite-tools-win32-x86-XXXXXXX.zip`
   (внутри будут файлы `sqlite3.exe` и другие инструменты).
Название архива может немного отличаться по версии, но главное — чтобы в нём были инструменты (`sqlite-tools`).
2. Распаковать архив

1. Найдите скачанный файл `.zip` в папке `Загрузки`.
2. Кликните по нему правой кнопкой → **Извлечь всё...**.
3. Выберите, куда распаковать (например: `C:\sqlite`).
В итоге у вас должна появиться папка, внутри которой есть файл:
- `sqlite3.exe`

3. Запустить sqlite3 через командную строку
1. Нажмите **Win + R** → введите `cmd` → нажмите Enter. Откроется **Командная строка**.
